ساختار پایگاه‌داده نمایندگی‌های خودرو

این صفحه ساختار پایگاه‌داده نرمال‌سازی‌شده تا سطح 4NF را نمایش می‌دهد.

جداول اصلی

جدول نمایندگی‌ها (Dealerships)

فیلد نوع داده توضیحات
DealershipID INT کلید اصلی
Name VARCHAR(100) نام نمایندگی
Address VARCHAR(255) آدرس نمایندگی
ManagerName VARCHAR(100) نام مدیر نمایندگی

جدول پرسنل (Employees)

فیلد نوع داده توضیحات
EmployeeID INT کلید اصلی
Name VARCHAR(100) نام کارمند
Position VARCHAR(100) سمت کارمند
DealershipID INT کلید خارجی (مرجع به نمایندگی‌ها)

جداول ارتباطی

جدول کارمند-بخش (Employee-Department)

فیلد نوع داده توضیحات
EmployeeID INT کلید خارجی (مرجع به پرسنل)
DepartmentID INT کلید خارجی (مرجع به بخش‌ها)
